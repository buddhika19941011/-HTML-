# -HTML-
"සිංහල HTML වැඩබිම" යනු ඉංග්‍රීසි භාෂාවට අමතරව මව්බසින්ම වෙබ් පිටු නිර්මාණය කිරීමට සිංහල කතා කරන පරිශීලකයින්ට ඉඩ සලසන නව්‍ය වෙබ් යෙදුමකි. ඔබ ලියන සිංහල කේත ක්ෂණිකව සම්මත HTML බවට පරිවර්තනය කිරීමට මෙය සමත් වේ.  ඔබ වෙබ් සංවර්ධනයට අලුතින් එක්වන අයෙකු නම් හෝ මව්බසින් කේත ලිවීමට කැමති අයෙකු නම්, මෙම යෙදුම ඔබට කදිම විසඳුමකි!
